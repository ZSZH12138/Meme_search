Meme Search

一个基于剪贴板监听与快捷键触发的本地表情包搜索工具。

通过快捷键将文本发送至后台，自动匹配并返回最合适的表情包图片。



安装依赖

请确保已安装 Python（建议 Python 3.11）。



在项目根目录下运行：

pip install -r requirements.txt



运行方式

在项目根目录执行：

python main.py



重要：

请务必等待电脑右下角出现提示：

程序已准备完成

只有看到该提示后，才可以开始操作。



使用流程



第一步：复制目标文本

使用：

Ctrl + C

将你想匹配表情包的文本复制到剪贴板。



第二步：发送文本到后台

使用快捷键：

Ctrl + Shift + E

程序会将剪贴板中的文本发送至后台。

系统会在右下角提示：

已复制文本：xxx 或 未检测到文本（如果剪贴板为空）



第三步：选择表情包

后台会弹出选择框。

请选择你需要的表情包。



第四步：粘贴图片

选定后：

图片会自动复制到剪贴板

右下角会提示：

图片已粘贴至文本框

此时你只需：

Ctrl + V

即可将表情包发送出去 



提示说明

整个使用过程中，电脑右下角会显示状态提示，包括：

请留意提示信息以确保操作顺利。



注意事项

必须等待程序初始化完成

剪贴板必须包含文本内容

快捷键冲突时请检查系统热键设置

请勿关闭后台进程，否则快捷键功能失效
