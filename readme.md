# Meme Search

一个基于剪贴板监听与快捷键触发的本地表情包搜索工具。通过快捷键将文本发送至后台，自动匹配并返回最合适的表情包图片。  

如果想要使用代码运行 可下载V1.0.0版本后，从release中复制raw_data文件夹到当前目录下，运行main文件即可。

---

## 安装依赖

请确保已安装 **Python**（建议使用 Python 3.11）。

在项目根目录下运行：
```bash
pip install -r requirements.txt
```

---

##  运行方式

在项目根目录执行：
```bash
python main.py
```
[请着重注意！！！]  
启动检查：请务必等待电脑右下角出现提示：“程序已准备完成”。只有看到该提示后，快捷键功能才正式激活。

---

## 使用流程

第一步：复制目标文本  
使用 Ctrl + C 将你想匹配表情包的文本（如“我想开了”、“流泪”）复制到剪贴板。  

第二步：发送文本到后台
使用快捷键 Ctrl + Shift + E。程序会将剪贴板中的文本发送至后台处理。  
系统会在右下角提示：已复制文本：xxx 或 未检测到文本（如果剪贴板为空）。  
  
第三步：选择表情包  
后台会弹出表情包选择框，请从中点击选择你最心仪的那张。  

第四步：粘贴图片  
选定后，图片会自动复制到剪贴板，右下角提示：图片已粘贴至文本框。  
此时只需在聊天窗口使用 Ctrl + V 即可将表情包发送出去。

---

## 数据处理说明
项目中的 data_process 文件夹负责对原始图片数据进行预处理，确保搜索功能的准确与高效：  

元数据提取：获取图片文件名及其对应的链接，并汇总到 json 文件中。  

语言优化：自动将图片名从繁体中文转化为简体中文，提升搜索匹配率。  

向量化建模：将图片数据 Embedding 成向量 npy 文件和 index 索引文件。  

产出位置：处理完成后，所有索引文件将统一存放于 data_index 文件夹中，供主程序调用。  

---

## 注意事项
初始化：必须等待程序初始化完成的通知，否则快捷键无效。  

剪贴板内容：触发快捷键前，请确保剪贴板内确实包含文本内容。  

快捷键冲突：若 Ctrl + Shift + E 无反应，请检查系统热键或其他软件（如通讯工具、IDE 等）是否占用了该组合键。  

后台常驻：使用期间请勿关闭后台 Python 进程，否则功能将立即失效。
